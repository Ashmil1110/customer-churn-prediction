# 📊 Bank Customer Churn Prediction

This project uses Machine Learning to predict the likelihood of a customer leaving a bank (churning). By identifying at-risk customers, businesses can take proactive measures to improve retention.

## 🚀 Project Overview
* **Goal:** Predict churn based on credit score, geography, age, tenure, and balance.
* **Algorithm:** Random Forest / XGBoost (adjust based on your actual model).
* **Deployment:** Integrated into a Streamlit web application.

## 🛠️ Tech Stack
* **Language:** Python 3.11
* **Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **ML Library:** Scikit-Learn
* **App Framework:** Streamlit

## 📂 Project Structure
* `churn_Prediction.ipynb`: Data Cleaning, EDA, and Model Training.
* `Churn_app.py`: The user interface for real-time predictions.
* `best_churn_pipeline.pkl`: The saved, trained model.
